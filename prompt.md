## 文体ルール

### 基本トーン

- 丁寧語（です・ます調）を基本とする
- 一人称は「私」または「わたしたち」（ブランド方針に合わせて統一）
- 読者への呼びかけは「あなた」を使う（「皆様」はフォーマルな場面向け）

### 文章の書き方

| ルール | 記入例 |
|--------|--------|
| 一文は短めに | 1文60字以内を目安にする |
| 専門用語は噛み砕く | 「コンバージョン」→「お問い合わせにつながる行動」 |
| 具体例を入れる | 抽象的な説明のあとに、場面が想像できる例を1つ添える |
| 見出しで構造化する | H2で大見出し、H3で小見出し。1見出しあたり2〜4段落 |

### 避ける表現

- 過度な断定（「絶対に」「必ず成功する」など）
- 競合の批判・比較での貶し
- 読者を責める表現（「それでもできないのはあなたのせい」など）

### 推奨する構成パターン

1. 導入：読者の悩みに共感する
2. 本論：原因の整理 → 解決の考え方 → 具体策
3. まとめ：要点の再確認
4. CTA：次の一歩を自然に促す
